Sobre Mim

Senior Data Engineer especializado em arquiteturas de dados em escala enterprise, com foco em Azure Databricks, Microsoft Fabric e Medallion Architecture.


Atualmente alocado na Petrobras via Compass UOL, onde mantenho SLA de 99,5% em pipelines ETL/ELT críticos, implemento governança com Delta Lake + Unity Catalog e entrego dados confiáveis como base para decisões executivas.

Minha especialidade é transformar dados fragmentados em ativos estratégicos, resolvendo os problemas que mais travam equipes: pipelines frágeis, governança inexistente e dados dispersos que ninguém confia.
